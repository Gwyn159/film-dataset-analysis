# MovieLens Analysis

Работа с датасетами MovieLens:

- ml-latest-small
- ml-latest

## Задание 1 Вариант 1

Анализ жанров для варианта:

- Animation
- Romance
- Documentary

Рассчитываются:

- количество фильмов по жанрам
- топ-10 фильмов по количеству рейтингов
- топ-10 фильмов с минимальным количеством рейтингов больше 10
- топ-10 фильмов с максимальным средним рейтингом при количестве рейтингов больше 10
- топ-10 фильмов с минимальным средним рейтингом при количестве рейтингов больше 10

## Задание 2 Вариант 2

Коллаборативная фильтрация по схожести объектов.

Рассчитываются:

- train/test split 80/20
- средний рейтинг в train_init
- baseline RMSE
- item-based collaborative filtering RMSE
